Для запуска программы необходимо:
* Python (https://www.python.org/downloads/)
* OpenPyXl - библиотека Python для работы с Excel (для установки напишите в консоли: pip install openpyxl)
* Excel

Для запуска программы напишите: python program.py
Либо запустите используя любой IDE, работающий с Python

Программа автоматически создаст файл Excel при выборе варианта [2] если файл "Рабочий файл.xlsx" отсутствует
